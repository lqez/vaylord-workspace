# vaylord-workspace

Public workspace for the **Vaylord** project.

This repository is public, so it serves as the central place for **issue
tracking and discussion** across the Vaylord projects. Issues for the
individual project repositories are filed and managed here.

## Why this repo?

The actual source code lives in separate repositories. This workspace exists to
give those projects a single, public home for reporting bugs, requesting
features, and coordinating work — without requiring access to the code repos
themselves.

## Projects tracked here

- **[vaylord](https://github.com/lqez/vaylord)** — *Arcana & Mesh*, a
  self-playing, autonomous-agent MMORPG. The world simulates itself on the
  server; characters are autonomous agents and you act as a patron, shaping the
  story with directives and resources.
- **[vaylord-world-creator](https://github.com/lqez/vaylord-world-creator)** — a
  web-based world **generation + editing** tool for Vaylord. Procedurally
  generate continents in the browser, sculpt heightmaps and biome maps, and
  export them in a format Vaylord can consume directly.

## Reporting issues

Please open issues in **this** repository, regardless of which project they
relate to. To help us route and triage:

- **Title** — start with the affected project, e.g. `[vaylord]` or
  `[world-creator]`.
- **Description** — include clear steps to reproduce, expected vs. actual
  behavior, and environment details (OS, browser, version/commit) where
  relevant.
- **Type** — note whether it's a bug report, feature request, or question.

Thanks for helping improve Vaylord!
