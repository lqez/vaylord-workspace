# VayLord 세계관 정전(Canon) — 운영 규칙

이 디렉터리는 VayLord 세계관의 **단일 소스(SSOT)** 입니다. 지명·천체·세력 등
고유명은 `labels.csv` 한 곳에만 두고, 지도(카토그래피), world-creator의 POI/코덱스,
게임(vaylord)이 모두 이 파일을 소비합니다. 라벨은 이미지에 굽지 않고 오버레이로
표기합니다(재생성 없이 수정 가능).

> 출처: OpenMMO WORLD_BUILDING.md 의 운영 방식을 연구해 채택
> (lqez/vaylord-workspace#53). 짧고 명확하게, 현재진행형으로 씁니다.

## 확장 체크리스트 (설정 추가 전 4단계)

1. **충돌 확인** — `labels.csv`·기존 정전 문서와 명칭/세력/연표가 충돌하지 않는가?
2. **기능+서사** — 기계적 기능(코드에서 뭘 하나)과 서사적 목적을 둘 다 한 줄씩 쓸 수 있는가?
3. **체감 확인** — 플레이어(또는 에이전트)가 실제로 마주치는 설정인가?
4. **기록** — `CHANGELOG.md`에 버전 항목을 남겼는가?

## 명명 우선순위 규칙

- **정전 고유명**(이 파일)은 *종류/조건 수준*에만 부여한다: 천체, 프로빈스(기후 대구획),
  혈통, 세력, 역사적 사건. 좌표가 아니라 **조건**으로 기술한다
  (예: "가장 큰 Martian 프로빈스" — 시드가 달라도 성립).
- **개별 거점/지형지물 이름**은 world-creator가 시드로 생성한다(settlementNames).
  정전명과 생성명이 충돌하면 **정전명이 이긴다**.
- 디버그/메타 도구(배속, 캡처 모드 등)는 정전이 아니다.

상대 표기한다. 천체 주기(4위성 1/12/…개월, 96달 대정렬)는 코드가 곧 정전이다
